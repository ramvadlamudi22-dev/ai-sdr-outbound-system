# AI SDR Outbound System

Prototype scaffold for an AI-assisted business review and follow-up workflow.

## Status

MVP planning, documentation, sample data, prompt notes, and starter scripts are included.

## Why this exists

Many small teams collect leads, requests, and business records in messy places. This repository explores a simple assistant-style workflow that helps organize records, prepare summaries, and keep next steps clear.

## Core Flow

```text
Input Records -> Review Notes -> Priority -> Status -> Report
```

## What is included

- project overview docs
- architecture notes
- sample records
- prompt template
- starter Python script
- demo report format
- open roadmap issues

## Human Review Policy

This project is designed for human-reviewed assistance. It does not send messages automatically. Suggested notes and next steps should be checked by a person.
